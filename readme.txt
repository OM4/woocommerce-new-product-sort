When adding a new WooCommerce product, automatically move it to the last item in the drag & drop sort order.

== Description ==

In WooCommerce, if using drag & drop product sorting, when a new product is added it will show at the beginning of the list by default.

This plugin ensures that new products created get added to the end of the product sort order by default.

If you have any other suggestions for other information that could be included, please [add your suggestion via GitHub](https://github.com/OM4/woocommerce-shipping-always/issues).

== Installation ==

1. Download the latest stable version of the plugin from ttps://github.com/OM4/woocommerce-new-product-sort
1. Install this plugin via WordPress Dashboard, Plugins, Add New, Upload.
